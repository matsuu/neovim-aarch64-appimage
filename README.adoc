= neovim-aarch64-appimage

Build nvim.appimage for aarch64(arm64) architecture

Since v0.10.4, Neovim has been distributing an AppImage for arm64, making this repository obsolete.

== Requirements

* Docker

== Usage

build master branch

----
./build.sh
----

build with specified version

----
./build.sh v0.7.2
----
